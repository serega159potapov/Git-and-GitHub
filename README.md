# HTML Проект - Урок 8

Этот проект представляет собой простую HTML страницу с базовыми стилями CSS.

## Описание

Проект содержит:
- `index.html` - основная HTML страница с встроенными CSS стилями
- `html-logo.svg` - иконка для favicon

## Структура проекта

```
├── index.html          # Основная HTML страница
├── html-logo.svg       # Иконка проекта
└── README.md          # Документация проекта
```

## Особенности

- Адаптивный дизайн
- Встроенные CSS стили
- Цветовая схема с акцентами
- Интерактивные элементы

## Как запустить

1. Откройте файл `index.html` в любом современном браузере
2. Или используйте локальный сервер для разработки

## Технологии

- HTML5
- CSS3
- SVG (для иконки)
